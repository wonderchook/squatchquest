## AI Platform Settings & Optimization

### Midjourney Settings
Use these parameters for all Midjourney prompts:
```
--ar 1:1 --s 50 --style raw --v 6.1
```

**Full prompt example for Midjourney:**
```
--ar 1:1 --s 50 --style raw --v 6.1 [base prompt text] /imagine prompt: Bigfoot sniffing a single large flower with curiosity,
flat vector illustration, bold simple shapes, minimal detail, high contrast, solid color fills, no texture, no gradients,
thick outlines, poster style, 1080x1080 pixels, square format, only 2-3 colors, designed to work as a QR code background
```

**Key settings explained:**
- `--ar 1:1` = Square aspect ratio (1080x1080)
- `--s 50` = Moderate stylization for clean, bold vector look
- `--style raw` = Removes Midjourney's default artistic processing for cleaner output
- `--v 6.1` = Uses version 6.1 model for best vector illustration quality

### DALL-E 3 Settings
DALL-E 3 works best with detailed natural language prompts. Include in your text:

**Recommended approach:**
- Paste the full prompt text (without Midjourney parameters)
- Select "Standard" quality for faster generation
- Use "1024x1024" as output size, then resize to 1080x1080
- Keep "Style" set to "Vivid" for high contrast
- Use "Natural" generation mode for best vector-like appearance

**Note:** DALL-E 3 may add more details than desired. Use image editing to simplify after generation.

### Stable Diffusion
Use Stable Diffusion XL (SDXL) for best results with vector illustrations.

**Recommended parameters:**
- **Model:** Stable Diffusion XL
- **Steps:** 25-30 (for clean, simple outputs)
- **Guidance Scale:** 7-8 (balances prompt adherence with quality)
- **Sampler:** DPM++ 2M Karras
- **Aspect Ratio:** 1:1 (square)
- **Resolution:** 1024x1024 (resize to 1080x1080 in post-processing)

**Negative prompt to include:**
```
gradients, textures, photorealistic, detailed, shading, shadows, 3D, complicated,
busy background, multiple objects, small details
```

## QR Code Integration Methods

### Method 1: QuickQR.art Integration (Recommended)
QuickQR.art allows seamless QR code overlays on your generated images.

**Steps:**
1. Generate your image using Midjourney, DALL-E 3, or Stable Diffusion
2. Download the generated image (1080x1080)
3. Visit [quickqr.art](https://quickqr.art)
4. Click "Create QR Code"
5. Enter your destination URL (business website, Google Maps link, etc.)
6. Select "Upload Background Image"
7. Upload your Bigfoot illustration
8. Adjust QR code positioning (typically centered or bottom-right)
9. Adjust QR code opacity (20-40% for readability while preserving background)
10. Click "Download" to save final image

**Advantages:**
- No special software needed (web-based)
- QR code automatically optimized for scanning
- Can easily adjust transparency and position
- Maintains image quality

### Method 2: Manual Overlay (Photoshop / GIMP)
For more control over QR code placement and styling.

**Steps:**
1. Generate your AI image (1080x1080 pixels)
2. Generate a QR code using a standard QR code generator with white background
3. Open your AI image in Photoshop or GIMP
4. Create a new layer for the QR code
5. Import or paste the QR code onto this layer
6. Resize QR code to occupy 20-30% of the image (typically 200-300 pixels)
7. Position in corner or center (typically bottom-right or center)
8. Adjust layer opacity to 70-85% for visibility
9. Flatten image and export as PNG

**Recommended QR Code Generators:**
- [QR Code Generator](https://www.qr-code-generator.com/)
- [QR Stuff](https://www.qrstuff.com/)
- [GoQR.me](https://goqr.me/)

### Method 3: Figma Vector Overlay
For a more polished, integrated look.

**Steps:**
1. Create a 1080x1080 pixel artboard in Figma
2. Import your AI-generated background image
3. Generate QR code as vector or high-res PNG
4. Import QR code into Figma
5. Layer and position as needed
6. Export as PNG at 2x resolution (2160x2160), then scale down to 1080x1080

## Generation Tips & Best Practices

### Achieving Optimal Vector Look
- If AI results are too detailed, post-process using:
  - Photoshop's "Posterize" filter (3-4 levels)
  - GIMP's "Simplify" or "Cartoon" filter
  - Illustrator's "Image Trace" feature

### Color Matching
- Export generated images with exact hex color backgrounds
- If background color doesn't match exactly, use color replacement tools:
  - Photoshop: Select Color Range > Replace Color
  - GIMP: Select by Color, then fill with exact hex code
  - Online: Remove.bg or similar color replacement tools

### QR Code Readability
- Test all final images with a QR code reader before deployment
- Ensure sufficient contrast between QR code and background
- Minimum QR code size: 200x200 pixels on 1080x1080 canvas
- Maximum QR code size: 40% of canvas (432x432 pixels)

### Quality Assurance Checklist
- [ ] Image is exactly 1080x1080 pixels
- [ ] Background color matches hex code from color reference table
- [ ] Illustration contains only 2-3 colors
- [ ] No gradients, textures, or fine details
- [ ] Thick outlines on all shapes
- [ ] QR code is centered or positioned consistently
- [ ] QR code scans correctly to destination URL
- [ ] Image file size is optimized (under 500KB)
